רמה 2 — הוצאת גורם משותף (x ו־x²)

פתור/י באמצעות העברת אגפים במידת הצורך והוצאת גורם משותף. אין להשתמש בנוסחת שורשים.

\(x^2 + 3x = 0\)
\(x^2 - 7x = 0\)
\(-4x = -4x^2\)
\(2x^2 + 10x = 0\)
\(5x^2 - 15x = 0\)
\(3x^2 = 12x\)
\(7x = x^2\)
\(9x^2 + 6x = 0\)
\(x^2 + x = 0\)
\(4x^2 - x = 0\)
\(6x^2 + 3x = 0\)
\(2x^2 = -8x\)
\(10x = -5x^2\)
\(x^2 - 12x = 0\)
